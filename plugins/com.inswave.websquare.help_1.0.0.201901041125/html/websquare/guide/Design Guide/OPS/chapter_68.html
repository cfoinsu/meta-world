59.SlideHide

수정 일자 버전 (SP2)

  • WebSquare5 Studio : 20180117_1349_x86_B

  • WebSquare5 Engine : 5.0_2.2982B.20180223.154326_1.5

SlideHide 는 이벤트에 따라 특정 layer를 animation으로 보였다가 사라졌다 할 때 사용하는 컴포넌트 입니다. 퀵 메뉴와 같이 mouse를 올렸을때 잠시 나타나거나 할때 등의 상황에서 사용합니다.

특정 영역을 사용자가 지정한 이벤트(click, mouseOver)가 발생하면 좌, 우 또는 상,하 중 사용자가 지정한 방향으로 영역이 슬라이드 되면서 보이거나 숨김처리가 가능한 기능입니다.

59.1Property

Property

Description

class

사용자가 임의의 class를 정의할때 사용합니다.

  • firstStatus

처음에 보여질지 숨겨져 있을지 여부를 설정합니다.

시작될 형태(hide or slide)

  • hideEvent

hide가 발생할 이벤트의 종류를 지정합니다.

hide가 발생할 이벤트(onmouseout, onmouseover, onclick, ondblclick)

id

사용자가 임의의 id를 정의할때 사용합니다.

  • moveSize

애니메이션시 한번에 움직일 단위입니다. 높을수록 애니메이션 프레임이 저하되며 속도는 빨라집니다. 반대로 단위가 낮으면 애니메이션 프레임은 높아지며 속도는 느려집니다.

슬라이드될 사이즈

  • showSize

hide 되고난 후 보여지는 크기를 지정합니다.

슬라이드될 사이즈

slideDirection

slide방향을 지정합니다.슬라이드되어 보여질 방향(right, left, up, down)

  • slideEvent

slide가 발생할 이벤트의 종류를 지정합니다.

slide가 발생할 이벤트(hideEvent와 같음)

tabIndex SP2

Tab Key를 이용 한 컴포넌트의 포커스 순서로 HTML의 tabindex속성과 동일한 기능을 제공한다.

59.3CSS Overriding

[그림 59-2]02

No

Class

Description

1

w2slideHide, slide_hide

w2slideHide는웹스퀘어 엔진에서 정의된 클래스입니다, 기본적으로 html 로 떨구는 시점에서 자동으로 부여됩니다. SlideHide컴포넌트를 숨김처리 하기 위하여 추가된 부모 객체입니다. 슬라이드 동작시 해당 객체의 너비, 높이값이 변하면서 슬라이드 동작이 이루어집니다.slide_hide는 사용자 정의 클래스로 Stylesheet_ext.css 다음에 적용됩니다. 동일한 스타일 시트를 정의함으로써 overriding(덮어쓰기) 됩니다.

2

w2slideHide_slideHideMain

w2slideHide_slideHideMain는 웹스퀘어 엔진에서 정의된 클래스입니다, 기본적으로 html 로 떨구는 시점에서 자동으로 부여됩니다. SlideHide컴포넌트의 실제 영역입니다.사용자 클래스로 접근할 경우에는 .slide_hide .w2slideHide_slideHideMain{}이런식으로 접근하여 overriding(덮어쓰기) 합니다.
