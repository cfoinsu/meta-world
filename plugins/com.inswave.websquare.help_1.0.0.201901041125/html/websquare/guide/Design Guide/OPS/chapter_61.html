모서리가 원형 처리된 사각형 공간을 생성합니다. 기본적으로 SVG로 생성되며 익스플로러 9이하의 브라우저에서는 VML(XML기반의 익스플로러 전용 웹그래픽 언어)로 랜더링 처리됩니다.
