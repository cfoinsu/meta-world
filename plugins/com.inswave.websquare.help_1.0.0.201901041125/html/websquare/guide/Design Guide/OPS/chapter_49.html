mainMenu와 그 mainMenu의 하위로 childMenu를 구성하는 일반적인 메뉴 컴포넌트입니다. 가로메뉴와 세로메뉴, 2가지 종류의 메뉴로 표현이 가능합니다. Style의 class를 지정하여 원하는 형태로 메뉴 스타일을 변경할 수 있으며, 이벤트를 이용하여 메뉴가 선택되었을 때의 액션을 취할 수 있습니다.

40.1Property

Property

Description

class

사용자가 임의의 class를 정의할때 사용합니다.

dataType

메뉴의 데이터 타입을 설정합니다.

recursive, listed로 2가지가 있습니다. recursiverecursive는 label과 value 정보를 가진 특정 노드 a의 하위노드로하나 이상의 같은 형식의 a노드가 반복적으로 구성된 XML형식입니다.반복적인 XML이 depth를 표현하고 있으므로 별도의 depth 정보를 필요로 하지 않습니다.

<recursive의 code 형식>


listed

listed는 label과 value 정보를 가진 특정 노드 a가 같은 레벨에 하나 이상의 노드로 나열된 XML형식입니다.노드 정보가 동일 레벨에 있으므로 별도의 depth 정보가 필요합니다.

<listed의 code 형식>

firstMenuGap

menu와 menu 사이에 여백의 크기를 지정합니다.

id

사용자가 임의의 id를 정의할때 사용합니다.

isVertical

true인 경우 메뉴를 세로로 표시합니다.

keepFirstSelectedMenu

메뉴 이벤트 종료 후 1Depth 메뉴의 스타일 유지 여부를 지정합니다.

menuHide

마우스가 메뉴 밖으로 완전히 나간 경우에 하위메뉴들을 닫을지 지정합니다.

menuWidthType

menu의 width를 어떻게 정할지 결정합니다.

overlapHeight

하위 노드와 부모노드를 부분적으로 겹치게 할 height.

overlapWidth

하위 노드와 부모노드를 부분적으로 겹치게 할 width.

selectedMenuIndex

첫 로딩시부터 클릭되어져 있도록 css를 적용할 메뉴 인덱스, 0부터 시작합니다.

showChildMenu

하위메뉴를 보여주는 이벤트를 어떤것으로 할지 지정합니다.

tabIndex SP2

Tab Key를 이용 한 컴포넌트의 포커스 순서로 HTML의 tabindex속성과 동일한 기능을 제공한다.

wmode

메뉴 wmode 옵션을 지정합니다.

No

Class

Description

1

w2menu,menu_comp

w2menu는 웹스퀘어 엔진에서 정의된 클래스입니다, menu_comp는 사용자 정의 클래스로 Stylesheet_ext.css 다음에 적용됩니다. 동일한 스타일 시트를 정의함으로써 overriding(덮어쓰기) 됩니다.menu컴포넌트의 바깥을 감싸는 최상위 객체입니다.

2

w2menu_main

w2menu는 웹스퀘어 엔진에서 정의된 클래스입니다, overriding(덮어쓰기)시 사용자정의 클래스를 이용하여 .menu_comp .w2menu_main{}이런식으로 overriding(덮어쓰기)하여 사용합니다.각각의 메뉴들을 감싼 부모 객체입니다.

3

w2menu_node, mainMenuDefault

w2menu_node, mainMenuDefaultw2menu는 웹스퀘어 엔진에서 정의된 클래스입니다, 각메뉴의 사각형 영역입니다

4

w2menu_label

w2menu_label는 웹스퀘어 엔진에서 정의된 클래스입니다, 각 메뉴의 글자를 감싼 영역입니다.

40.4주의사항 및 Tip

  • 메뉴컴포넌트는 내부시스템이나 웹표준에 대한 요구가 없을 경우 사용권장합니다.

  • 웹스퀘어에서는 메뉴컴포넌트보다 group컴포넌트를 이용, css 및 javascript 등을 활용한 웹표준방식을 권장 해 드립니다.
