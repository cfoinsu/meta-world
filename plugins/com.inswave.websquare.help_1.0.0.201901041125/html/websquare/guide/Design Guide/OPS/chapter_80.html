71.Upload

Upload는 Input타입으로 구현되어 제공 되고 있으며, 한개의 파일을 업로드 할 수 있는 컴포넌트 입니다.

Upload컴포넌트는 image와 button의 두가지의 속성을 가지고 있습니다. Image는 input과 버튼이 결합된 형태로 만들어져 있으며, input Type=”text”을 사용하며 button은 브라우저의 형태이며, input Type=”file”을 사용한다.

71.1Property

Property

Description

accept

IE10부터 지원되는 기능. <input type="file"/> 에서 accept 속성과 동일한 기능을 제공한다. 사용 방법은 http://www.w3schools.com/tags/att_input_accept.asp 참

(ex) // jpg, png, xml, txt 파일만 선택할 수 있도록 OS fileManager에 요청한다.

주의사항 : file validation은 반드시 서버에서 별도로 체크할것. 즉 이 속성을 file vaildation목적으로 사용해선 안 된다.
