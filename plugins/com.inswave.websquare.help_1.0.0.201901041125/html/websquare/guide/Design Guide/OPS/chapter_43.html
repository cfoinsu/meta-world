34.Header (HTML5)

<header> 태그는 문서 또는 문장의 머리글을 작성하기 위한 태그 입니다. header는 일반적으로 내용소개 또는 네비게이션 링크를 포함하고 있습니다.

Property

Description

컴포넌트의 ID로 전역객체로 할당되며 script에서 본 id로 컴포넌트에 접근이 가능하다.
