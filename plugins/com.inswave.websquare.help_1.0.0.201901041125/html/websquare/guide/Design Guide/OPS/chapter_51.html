42.Multiupload

수정 일자 버전 (SP2)

  • WebSquare5 Studio : 20180117_1349_x86_B

  • WebSquare5 Engine : 5.0_2.2982B.20180223.154326_1.5

Multiupload는 flash로 구현되어 제공 되고 있으며, 여러 개의 파을을 동시에 업로드 할 수 있는 컴포넌트이다. 또한 투명모드(ode:transparent)를 지원함으로써 사용자가 원하는 UI를 만들 수 있다.

Flash로 되어있어서 웹접근성이나 모바일에서는 사용할 수 없습니다.

42.2브라우저 캡쳐화면

Upload컴포넌트는 image와 button의 두가지의 속성을 가지고 있습니다. Image는 input과 버튼이 결합된 형태로 만들어져 있으며, input Type=”text”을 사용하며 button은 브라우저의 형태이며, input Type=”file”을 사용한다.

42.4주의사항 및 Tip

multiUpload 컴포넌트의 mode="transparent" 를 사용, 선택한 파일목록을 그리드로 구현

  1. 버튼 위에 mode="transparent" 속성의 multiupload 컴포넌트를 위치한다.

  2. 버튼 클릭시 투명모드의 multiupmoad컴포넌트의 추가 버튼클릭 이벤트가 발생한다.
