60.Slider

수정 일자 버전 (SP2)

  • WebSquare5 Studio : 20180117_1349_x86_B

  • WebSquare5 Engine : 5.0_2.2982B.20180223.154326_1.5

Slider 는 상하, 좌우로 슬라이드 되는 형태의 바로 표현되며, 버튼을 드래그 해서 값을 선택하는 ui를 만드는 컴포넌트 입니다.

60.1Property

roperty

Description

class

사용자가 임의의 class를 정의할때 사용합니다.

id

사용자가 임의의 id를 정의할때 사용합니다.

increment

슬라이더의 증가값의 단위를 지정합니다.

maxValue

slider의 최대값을 지정합니다.

minValue

slider의 최소값을 지정합니다.

ref

model instance의 바인딩 정보를 입력합니다.

setBackground

그래그한 범위만큼 바를 배경색으로 채웁니다.

showScale

slider의 상하에 눈금을 표시할지 여부를 지정합니다.

tabIndex SP2

Tab Key를 이용 한 컴포넌트의 포커스 순서로 HTML의 tabindex속성과 동일한 기능을 제공한다.

title

웹접근성을 위한 속성으로서 컴포넌트의 설명글을 입력합니다.

tooltipDisplay

slider의 값을 slider버튼위에 보여줍니다.

value

초기 값을 설정합니다.

vertical

세로 슬라이더를 사용할지 지정합니다.

w2slider_startLayer는 웹스퀘어 엔진에서 정의된 클래스입니다, 기본적으로 html 로 떨구는 시점에서 자동으로 부여됩니다.Slider의 왼쪽 배경 영역입니다.

w2slider_centerLayer

w2slider_centerLayer는 웹스퀘어 엔진에서 정의된 클래스입니다, 기본적으로 html 로 떨구는 시점에서 자동으로 부여됩니다.Slider의 가운데 여역으로서 실제 drag되는 영역입니다..

4

w2slider_endLayer

w2slider_endLayer는 웹스퀘어 엔진에서 정의된 클래스입니다, 기본적으로 html 로 떨구는 시점에서 자동으로 부여됩니다.Slider의 오른쪽 배경 영역입니다.

5

w2thumb

w2thumb는 웹스퀘어 엔진에서 정의된 클래스입니다, 기본적으로 html 로 떨구는 시점에서 자동으로 부여됩니다.Slider의 bar부분 입니다.

60.4주의사항 및 Tip

  • onthumbmove 이벤트는 슬라이더 컴포넌트의 핸들을 이동하며 동시에 발생되는 이벤트 입니다.

  • onmovedone 이벤트는 슬라이더 컴포넌트의 핸들을 이동하고 난 후 발생되는 이벤트 입니다.
