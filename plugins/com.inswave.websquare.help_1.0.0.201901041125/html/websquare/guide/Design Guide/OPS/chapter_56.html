47.PageList

데이터양이 많은 그리드를 사용할때 유용합니다.(페이지 수와 페이지를 선택)

PageList는 페이징 처리가 필요한 경우 multi-page를 표현하기 위해 제공 되는 사용자 인터페이스 컴포넌트 입니다. 기본적으로 사용할 때 웹스퀘어에서 제공하는 API를 사용해야 정상작동하는 컴포넌트입니다.

47.1Property

Property

Description

  • buttonShowType

페이지 이동 버튼의 타입 (default는 0)

  • 0 : 이전 목록 페이지, 다음 목록 페이지 버튼으로 구성

  • 1 : 이전 목록 페이지, 이전 페이지, 다음 페이지, 다음 목록 페이지 버튼으로 구성

  • 2 : 이전 페이지, 다음 페이지 버튼으로 구성

  • 3 : 첫 페이지, 이전 목록 페이지, 다음 목록 페이지, 마지막 페이지 버튼으로 구성

  • deliImage

구분자의 이미지

  • deliWidth

구분자의 가로갈이

  • delimiter

구분자

  • displayButtonType

첫페이지 또는 마지막 페이지시 페이지버튼 처리

  • display : 항상표시

  • auto : 마지막 페이지엔 표시안함

  • displayOver : pageCount가 pageSize보다 클때 표시(작을 경우엔 display와 동일하게 동작)

  • autoOver : pageCount가 pageSize보다 클때 표시(작을 경우엔 auto와 동일하게 동작)

  • autoHidden : 첫 페이지, 마지막 페이지에 보여지는 수 만큼은 표시안함

displayFormat

index가 표시되는 Format

  • leftPageBackImage

페이지를 왼쪽으로 pageSize단위 만큼 바꾸는 버튼의 이미지

  • leftPageBackImageClass

페이지를 왼쪽으로 pageSize단위 만큼 바꾸는 버튼의 class

  • leftPageImage

페이지를 왼쪽으로 한칸 바꾸는 버튼의 이미지

  • leftPageImageClass

페이지를 왼쪽으로 한칸 바꾸는 버튼의 class

naviType

buttonShowType 2일 때만 가능한, 페이지 1개 이동관련 옵션 (pagingType은 무시)

  • oneMoveButtonWidth

페이지를 한칸 바꾸는 버튼의 width

pageSize

한 페이지에 보이는 숫자의 수

pagingType

남은 페이지의 수가 pageSize보다 작을 때 보는 방식

  • 1 : pagingSize에 맞춰서 표현

  • 2 : 남은 페이지만 표현

  • buttonShowType이 3인 경우에는 PagingType 1은 지원안함

  • renderType

렌더링되는 레이아웃 구조에 따라 "table", "list" 두가지 지원

  • table : table 구조로 구성되어 있음.

  • list : ul > li > a의 구조로 구성되어 있음.(웹접근성 준수시 꼭 renderType을 list로 해야됨)

  • rightPageBackImage

페이지를 왼쪽으로 한칸 바꾸는 버튼의 이미지

  • rightPageBackImageClass

페이지를 오른쪽으로 pageSize단위 만큼 바꾸는 버튼의 class

  • rightPageImage

페이지를 오른쪽으로 pageSize단위 만큼 바꾸는 버튼의 이미지

  • rightPageImageClass

페이지를 오른쪽으로 한칸 바꾸는 버튼의 class

  • shiftMoveButtonWidth

페이지를 pageSize단위 만큼 바꾸는 버튼의 width

useDelim

구분자 사용여부 , useDelim가 true 일때만 delimeter 사용가능

PageList를 보여주기 위한 기본 API - pageList.setCount();

  • PageList를 사용할경우 퍼블리싱할때는 기본적인 API를 사용해서 화면상에 보여주게 됩니다. 이때 사용하는 API가 setCount()라는 API입니다. 사용방법은 PageList(ID).setCount(100); 의 형태로 스크립트영역에 넣어주시면 됩니다. ()안의 100이라는 숫자를 넣어줄경우 100페이지까지 나타나게 된다.
