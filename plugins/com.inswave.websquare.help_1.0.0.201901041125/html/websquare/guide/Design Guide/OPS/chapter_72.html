63.Switch

수정 일자 버전 (SP2)

  • WebSquare5 Studio : 20180117_1349_x86_B

  • WebSquare5 Engine : 5.0_2.2982B.20180223.154326_1.5

Switch는 임의의 크기, 위치에 동적으로 화면 전환을 할 수 있는 컴포넌트 입니다. 특정 영역에 여러 개의 case로 화면을 구성하고 화면을 동적으로 변경할 수 있습니다.

tabControl과 유사한 기능을 수행하며, 탭버큰 목록 영역과 컨텐츠 페이지 영역으로 나뉘어져 각 탭버튼은 해당하는 컨텐츠 페이지로 바로 갈 수 있는 앵커를 제공합니다.

어떤 한 컨텐츠를 여러 페이지로 나누어 화면에 보여주는 페이징 기능이 필요할 때 사용하는 컨테이너성 컴포넌트이며, 자동으로 화면이 바뀔 때는 사용자가 이를 중지할수 있는 인터페이스를 제공해 주어야만 합니다.

63.3class 적용과 스타일 변경

마치 여러 개의 group 들이 겹쳐 있어 그 중 하나 만을 선택하여 보여주는 것과 같이 작동합니다. default로는 case1 만이 존재하고 case 추가를 통해 여러가지의 case 를 추가 할 수 있습니다.

Code 63-1사용방법

Property속성중에 slide속성을 true로 설정하고 Switch의 Click이벤트를 사용한다.

<xf:switch id="switch1" caseClass="" style="" class="xin"slide="true">

        <xf:group id="group2" style="width:326px;height:543px;" tagname="a">

                <w2:href>#</w2:href>
            </w2:attributes>
            <script ev:event="onclick" type="javascript"><![CDATA[

63.5주의사항 및 Tip

  • Switch 컴포넌트를 그리면 최초 1개의 case를 가집니다. Switch는 하나 이상의 case를 가질 수 있고, 화면을 동적으로 변경하는 기본적인 기능과 더불어 슬라이드 효과를 주어 가로 터치(브라우저:드래그 앤 드롭, 모바일:터치) 페이지 전환 기능을 사용할 수 있습니다.
