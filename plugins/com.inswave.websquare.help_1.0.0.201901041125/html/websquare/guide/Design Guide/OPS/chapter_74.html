65.TableLayout

수정 일자 버전 (SP2)

  • WebSquare5 Studio : 20180117_1349_x86_B

  • WebSquare5 Engine : 5.0_2.2982B.20180223.154326_1.5

TableLayout 컴포넌트는 Group 컴포넌트의 tagname속성에 HTML5 태그(<colgroup>, <tr>, <td>, <th>)를 추가하여 <table> 태그로 묶은 형태의 컴포넌트입니다.

표를 만들때 사용하는 컴포넌트

Property

Description

adaptive

tagname이 table일 경우 적응형 웹(다양한 화면크기와 디바이스에 맞게 레이아웃을 변경) 모드. 속성값이 crosstab인 경우, thead와 tbody가 각각 하나씩 있어야 함.

adaptiveThreshold

tagname이 table일 경우 adaptive가 설정되어 있을 경우 레이아웃 변경을 하기 위한 breakpoint(기본값 : 48

tabIndex SP2

Tab Key를 이용 한 컴포넌트의 포커스 순서로 HTML의 tabindex속성과 동일한 기능을 제공한다.

65.4편집방법

생성된 tableLayout에서 th 및 td를 선택한 후 엔터치시면 textbox처럼 직접 입력가능한 대화창이 나옵니다.

단순한 테이블구조를 그리려면 직접 th 및 td 태그에 텍스트를 입력해도 무방합니다. 하지만 th영역에 배경아이콘 및 추가적인 컴포넌트가 들어갈 때에는 textbox + 기타 컴포넌트 방식으로 문자열을 textbox안에 그리셔야 합니다. 아래에는 th태그에 textbox를 넣어서 문자열을 출력하는 방식을 설명해드리겠습니다.

65.5class 적용과 스타일 변경

[그림 65-7]5

컴포넌트창을 확인하시면 table태그, th 태그 및 td 태그에 자동으로 생성된 클래스가 보입니다.

이부분은 웹스퀘어 엔진에서 자동으로 파싱한 클래스여서 별도의 사용자 클래스를 생성해서 오버라이딩하는 방식으로 처리하는것을 권장합니다.

w2group과 w2tb는 웹스퀘어 엔진에서 정의된 클래스입니다, 기본적으로 html 로 렌더링되는 시점에서 자동으로 부여됩니다. 여기에 사용자 정의 클래스를 추가해서 overriding(덮어쓰기) 하여 디자인을 바꿀 수 있습니다.

65.7주의사항 및 Tip

  • Table 컬럼의 가로 값은 colgroup안 col의 style에 width="가로값" 을 수정해서 맞춰주시면 됩니다.

  • col의 갯수와 컬럼의 갯수는 같아야 됩니다.
