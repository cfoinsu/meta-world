모바일 사이트에 사용되는 스크롤이 가능한 컴포넌트입니다. - 모바일에서의 적용 ( iScroll.js 와 흡사 )

여러 다양한 컴포넌트를 그룹화 시킬 수 있는 컴포넌트 입니다.

Description

    z-index:20;
}

54.4CSS Overriding

[그림 54-3]3

No

Class

Description

1

w2scrollView, scroll_box

w2scrollView는 웹스퀘어 엔진에서 정의된 클래스입니다, 기본적으로 html 로 떨구는 시점에서 자동으로 부여됩니다. scroll_box 는 사용자 정의 클래스 입니다. 해당 클래스를 이용하여 디자인을 바꿀 수 있습니다.

2

scroll_box

사용자 정의 클래스로 Stylesheet_ext.css 다음에 적용됩니다. 동일한 스타일 시트를 정의함으로써 overriding(덮어쓰기) 됩니다.

3

.w2scrollView_iScrollX

가로 스크롤 입니다.

4

.w2scrollView_iScrollY

세로 스크롤 입니다.

54.5주의사항 및 Tip

  • 현재 요소로부터 position:relative; 속성을 가진 가장 가까운 (부모) 요소를 기준으로 스크롤 바를 생성합니다. ScrollView 자체에 스크롤을 생성하기 위해서는 ScrollView에 해당 style 속성을 지정하시면 됩니다.
