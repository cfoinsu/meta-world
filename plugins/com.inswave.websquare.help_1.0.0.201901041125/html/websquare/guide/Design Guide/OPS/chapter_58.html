49.Progressbar

수정 일자 버전 (SP2)

  • WebSquare5 Studio : 20180117_1349_x86_B

  • WebSquare5 Engine : 5.0_2.2982B.20180223.154326_1.5

HTML의 a Tag와 같은 기능을 제공하는 컴포넌트로 hyperlink를 정의할 수 있으며 CSS를 적용하여 버튼으로 사용합니다.

outerDiv 속성이 공백이거나 true일경우 최상위는 div Tag로 이루어 지며, outerDiv가 false일 경우 최상위는 a Tag 단독으로 이루어지며

또한 웹스퀘어에서 사용할 수 있는 버튼 컴포넌트중의 하나이며 버튼을 그릴 때 기본적으로 권장하는 컴포넌트입니다.

49.3class 적용과 스타일 변경

[그림 49-2]InputBox 디자인 적용 예제

먼저 progress 컴포넌트를 그린 후 클래스명을 mybar로 주겠습니다. 아래 화면을 보시면 Class에 w2progressbar라는 클래스명을 보실 수 있습니다. 이 클래스는 컴포넌트가 가지고 있는 고유의 클래스명이며. 기본 스타일이 정의되어 있습니다. w2progressbar이라는 클래스명을 사용하여 스타일을 주셔도 되지만, 그렇게 되면 모든 progressbar 스타일에 반영이 되니, 반드시 새로운 클래스를 부여하여 사용하시길 권장합니다.

[그림 49-3]2

mybar라는 클래스에 progressbar 의 배경색, 게이지 배경색, 말풍선 이미지 등을 지정해 주었습니다.

.mybar.w2progressbar.w2progressbar_type4 { 
    background: #f5f0f3;
    width:120px;
    height:30px;
}

.mybar .w2progressbar_bar { 
    background:-webkit-linear-gradient(#ef6a76 10%, #f5b5c4 11%, #fb6890 20%, #fb6890 80%, #ea3d65 89%, #de516d 90%);
    height: 50%;
    margin-top:7px; 
}

/* 말풍선 이미지 변경 */
.mybar .w2progressbar_bar div { 
    background:url(/DesignGuide/images_publishingTemplate/component/skyblue/progressbar_tooltip.png) no-repeat center bottom;
    top: -45px;
    right:-16px;
    width:32px;
    height:38px;
    line-height: 34px;
    font-size: 11px;
    text-align:center;
}
