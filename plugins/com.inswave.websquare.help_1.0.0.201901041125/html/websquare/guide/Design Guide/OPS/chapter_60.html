51.Repeat

수정 일자 버전 (SP2)

  • WebSquare5 Studio : 20180117_1349_x86_B

  • WebSquare5 Engine : 5.0_2.2982B.20180223.154326_1.5

컴포넌트 탬플릿을 통해 데이터 갯수만큼 자동 생성해주는 컴포넌트 입니다. 자동 생성시 id 및 ref를 data에 맞게 key 값으로 연동시켜줍니다.

51.1Property

Property

Description

autoCount

repeatKey속성을 반복되는 node에 설정 속성의 value는 처음노드 부터 1씩 커지면서 설정. (repeatKey가 존재하지 않을 경우, autoCount를 true로 하면 자동으로 설정.)

class

사용자가 임의의 class를 정의할때 사용합니다.

cols

반복되는 열의 횟수를 지정합니다.

direction

2차원 행렬의 형태로 node를 반복할 때 node의 반복되는 방향을 지정합니다.

id

사용자가 임의의 id를 정의할때 사용합니다.

nodeset

반복시킬 노드들의 xpath입니다.

repeatKey

노드의 value를 저장하는 변수입니다.

rows

반복되는 행의 횟수를 지정합니다.

tabIndex SP2

Tab Key를 이용 한 컴포넌트의 포커스 순서로 HTML의 tabindex속성과 동일한 기능을 제공한다.

toolTip

repeat의 toolTip 속성 글을 입력합니다.
