70.Trigger

수정 기준 버전 (SP2)

  • WebSquare5 Studio : 20180117_1349_x86_B

  • WebSquare5 Engine : 5.0_2.2982B.20180223.154326_1.5

Trigger는 type에 따라 이미지 형태의 버튼과 button 형태로 사용할 수 있으며,HTML에서<input>태그의 type=button과 유사한 인터페이스를 제공하는 컨트롤로 특정 Action과 컨트롤을 연계할 때 주로 사용되는 컴포넌트 입니다.

- button : input Tag의 type="button"으로 구성 된다.

- image : table Tag로 이루어져있고 left, center, right에 각각에 CSS 적용이 가능하다.(image 링크)

- anchor : 최상위 anchor Tag로 이루어져있고 button의 text는 span Tag로 구성된다.

70.1Property

Property

Description

centerOffImageClass

type속성의 값이 "image"일 경우만 적용되며 중앙 td에 적용될 class명을 정의한다.class에는 image의 url을 명시하며 일반적으로 width값은 제외한다.

centerOnImageClass

type속성의 값이 "image"일 경우만 적용되며 마우스 over 시 중앙 td에 적용될 class명을 정의한다.class에는 image의 url을 명시하며 일반적으로 width값은 제외한다.

class

HTML의 class속성과 동일한 기능을 제공하며 css파일 또는 style블럭에 정의 한 class를 1개 또는 공백(space)를 이용하여 다중 적용이 가능하다.

id

컴포넌트의 ID로 전역객체로 할당되며 script에서 본 id로 컴포넌트에 접근이 가능하다.

disabled

[default:false, true]HTML의 disabled속성과 동일한 기능을 제공하며 컴포넌트를 비활성화 시킨다.

leftOffImageClass

type속성의 값이 "image"일 경우만 적용되며 왼쪽 td에 적용될 class명을 정의한다. class에는 image의 url과 width 속성은 필수로 정의한다.

leftOnImageClass

type속성의 값이 "image"일 경우만 적용되며 마우스 over 시 왼쪽 td에 적용될 class명을 정의한다.class에는 image의 url과 width 속성은 필수로 정의한다.

nextTabID SP2

tab으로 이동시 이동할 컴포넌트 ID

rightOffImageClass

type속성의 값이 "image"일 경우만 적용되며 오른쪽 td에 적용될 class명을 정의한다. class에는 image의 url과 width 속성은 필수로 정의한다.

rightOnImageClass

type속성의 값이 "image"일 경우만 적용되며 오른쪽 td에 적용될 class명을 정의한다. class에는 image의 url과 width 속성은 필수로 정의한다.

style

InputBox의 style 속성

tabIndex

Tab Key를 이용 한 컴포넌트의 포커스 순서로 HTML의 tabindex속성과 동일한 기능을 제공한다

toolTip

웹스퀘어에서 만든 toolTip으로 컴포넌트의 추가정보를 명시한다. 마우스 over시 표현된다.

type

[default:button, image, anchor]Trigger 표현 방법을 정의하며 각 타입별 HTML 표현 방식은 아래와 같다.button : HTML input Tag로 표현되며 type="button"으로 구성 된다.image : 최상위 table Tag로 이루어져있고 left, center, right에 각각에 class적용이 가능하다.anchor : 최상위 anchor Tag로 이루어져있고 button의 text는 span Tag로 구성된다.

70.5disabled 일때의 css 처리

Property 창에서 disabled 속성을 true 또는 false를 줄 수 있습니다.

Disabled를 true로 줄 경우 버튼은 비활성화 상태로 표시 되어 클릭이 되지 않으며, disabled=””로 파싱 됩니다.

그리고 브라우저에서 파싱된 class를 보면 “w2trigger_disabled”로 나오는 것을 보실 수 있습니다.

disabled 일 때 “w2trigger_disabled” 클래스명에 아래와 같이 스타일을 주시면 됩니다.

70.7주의사항 및 Tip

  • type 속성을 이미지로 설정할 경우 table로 파싱이 됩니다.

  • type 속성을 이미지로 설정할 경우 배치성 테이블 형태로 (<summary>, <caption> 태그 제외됨) 사용 하여도 웹접근성에 문제가 되진 않지만

    명확한 웹접근성 준수를 위해 Property창의 class 속성에서 새로운 class명을 주어 input 스타일을 변경해 주는 방법을 권장합니다.
