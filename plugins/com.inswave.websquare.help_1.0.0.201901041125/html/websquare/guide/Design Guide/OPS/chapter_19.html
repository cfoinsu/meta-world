10.Canvas (HTML5)

HTML5 에서 새로 추가된 태그 입니다. canvas는 그래픽을 담기위한 컨테이너 역활을 합니다. 그래픽을 담기위해 스트립트를 작성해야 합니다.

10.1Property

Property

Description

id

컴포넌트의 ID로 전역객체로 할당되며 script에서 본 id로 컴포넌트에 접근이 가능하다.
