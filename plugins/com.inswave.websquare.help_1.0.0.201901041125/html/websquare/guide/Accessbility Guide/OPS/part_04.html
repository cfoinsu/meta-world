파트 IV.사용금지 컴포넌트

화면낭독기 사용자가 컴포넌트를 사용할수 있도록 구성할 방법을 제공하지 않는 컴포넌트를 의미합니다. 이들 컴포넌트 목록은 <표 4>와 같습니다.

<표 4>

컴포넌트명

사용금지 사유

activeX

W3C 표준이 아니며 사용되는 ActiveX 컴포넌트에 따라 접근성 정도가 달라집니다.

applet

<applet> 태그를 지원하는 브라우저가 거의 없는 수준이며 W3C 표준 태그가 아닙니다.

canvas

HTML5 속성으로 자바 스크립트 즉석에서 그래픽을 그리는 데 사용됩니다.

chart

웹스퀘어 기본 chart 컴포넌트는 접근성 지원에 많은 어려움이 있어 오픈소스인 jFreeChart와 image 컴포넌트를 적절히 연계해 사용하는 방안을 제공하고 있습니다

checkcombobox

접근성 지원을 하지 않는 컴포넌트입니다.

DatePicker

접근성 지원이 어려운 상황의 컴포넌트이며 모바일 전용 컴포넌트입니다.

flash

임베드할 플래시의 접근성 구현 정도에 의존해야 하며, W3C 표준이 아닙니다.

Gauge

접근성 지원이 어려운 상황의 컴포넌트입니다. 특화된 페이지에 적용됩니다.

menu

접근성 지원이 어려운 상황의 컴포넌트이며, tag 컴포넌트를 이용해 직접 구현하는 것을 권고합니다.

mutiSelect

접근성 지원이 어려운 상황의 컴포넌트이며, checkbox 컴포넌트로 대체해 사용하는 것을 권고합니다.

multiUpload

내부적으로 플래시를 사용하는 구조입니다. 여러개의 upload 컴포넌트로 대체해 사용하는 것을 권고합니다.

RoundedRectangle

사용안함

Slidehide

마우스 입력에 따라 숨기거나 보여지는 컨테이너성 컴포넌트입니다. Tab키 이동에 제한이 있습니다.

TreeView

기본적으로 키보드로 접근이 어렵기 때문에 사용이 어렵습니다. div/ul/li 형태로 만들어서 사용하세요.

windowContainer

현재 화면낭독기 기술로는 사용하는데 무리가 따르는 컴포넌트입니다
