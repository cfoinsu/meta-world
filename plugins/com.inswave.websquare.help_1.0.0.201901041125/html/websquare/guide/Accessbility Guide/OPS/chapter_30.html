28.1Tag

HTML 태그를 자유롭게 사용할 수 있도록 개발된 컴포넌트입니다. 이 컴포넌트를 이용해 모든 HTML 태그를 생성할 수 있지만, 웹스퀘어가 제공하는 다른 컴포넌트들을 이용해 생성할 수 없는 경우에만 사용하는 것을 권장합니다. 태그를 사용하고 속성을 넣을 경우는 Property창에서 html attributes속성을 이용해서 넣어도 되고, Source창에서 직접 html attributes속성을 넣어도 무방하다.

<Check Point>
