6.Calendar

6.1Calendar

달력을 보여주고 날짜를 선택하는 기능을 제공하는 컴포넌트이며, 접근성 지원을 위해선 renderType="native", keepFocusSelect="true", footerDiv="true" 로 설정하고, 달력의 의미에 맞는 명확한 title 속성과 caption, summary를 기술합니다

<Check Point>
√ renderType="native", keepFocusSelect="true", footerDiv="true"로 설정 되어있는가?
√ 키보드에 의한 초점이 논리적으로 이동하고 있는가?
√ 포커스 상태의 테두리(outline)가 명확하게 나타나는가?
√ 적절한 caption과 summary를 제공했는가? (html5 의 경우 caption만 제공)
관련 지침 : 2.1.1. 키보드 사용 보장, 2.1.2. 초점 이동, 3.2.1. 사용자 요구에 따른 실행, 3.3.2. 표의 구성

<Xml source>

<w2:calendar id="" style="" renderType="native" footerDiv="true" keepFocusSelect="true" summary="달력:원하는 날짜를 선택하세요." caption="달력"></w2:calendar>
