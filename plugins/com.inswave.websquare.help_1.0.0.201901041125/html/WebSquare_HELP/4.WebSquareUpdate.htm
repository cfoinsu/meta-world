4. WebSquare Update

1.2.1.1.2절 ‘WebSquare feature update”를 참조하여 귀사에 맞는 WebSquare Update Site를 등록합니다.

WebSquare Update 방법은 자동 update와 수동 update로 나뉩니다. 보다 나은 WebSquare Studio 환경을 위해 WebSquare feature Update를 자동 업데이트 설정하시길 권장합니다. 설정은 Eclipse를 실행하여 Menu bar의 Window à Preferences à Install/Update à Automatic Updates를 선택하여 다음의 화면 캡쳐와 같이 Automatically find new updates and notify me를 V표 체크하신 후 Apply 버튼이나 OK 버튼을 누르시면 됩니다.

 

수동 업데이트 방법은 Eclipse를 실행하여 Menu bar의 Help à Check For Updates를 선택하면 Update할 리스트가 있다면 다음의 화면 캡쳐의 Available Updates 창이 발생합니다.

 

Next 버튼을 눌러 다음 단계를 진행하면 최초 WebSquare feature Update와 같이 install창의 Available Software 내용이 출력될 것입니다. 이후 update 수행은 1.2.1.2절 ‘WebSquare feature update”의 내용과 동일합니다.
